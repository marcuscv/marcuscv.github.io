Online Auction System

This program implements a relational database system that simulates the operations of an online auction system. The auction system uses HTML for the user interface, Tomcat to host the web application, MySQL for the database server, and Java, Javascript, and JDBC to connect the user interface and the database server. Buyers and sellers can: create and delete accounts, login, and logout, post auctions, search and bid on auctions.

Pathfinding Visualizer

ReactJS web app that visualizes the behavior of Dijkstra’s algorithm in a grid between a starting node and a target node. The user can place block nodes in the grid, and the algorithm avoids these block nodes during its search. Highlights the shortest path if there is a path between the starting node and the target node.
